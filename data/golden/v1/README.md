# Golden data v1

This directory contains deterministic fixtures for regression testing.

- `inputs/`: canonical CSV inputs for thermodynamic analysis.
- `expected/`: reserved for generated expected JSON artifacts.

## Input schema

Each CSV in `inputs/` uses the repository's required results schema:

- `structure_id`
- `mechanism`
- `energy_eV`

## Included fixture scenarios

- `equal_energies.csv`: all energies identical.
- `dominant_emin.csv`: one clearly dominant minimum-energy state.
- `near_degenerate_minima.csv`: minima separated by micro-eV-level differences.
- `small_magnitude_energies.csv`: near-zero energies.
- `large_magnitude_energies.csv`: large-magnitude negative energies.
- `mixed_sign_energies.csv`: negative, zero, and positive energies.
- `realistic_small_sample.csv`: small realistic cluster of close energies.
- `two_state_split.csv`: compact two-state near-degenerate case.

Expected outputs for these inputs will be generated by `scripts/generate_golden.py` in a follow-up change.
