# Changelog

All notable changes to this project are documented in this file.

This project follows a lightweight semantic-ish versioning policy (`MAJOR.MINOR.PATCH`):
- `PATCH`: backwards-compatible fixes, docs updates, and maintenance.
- `MINOR`: backwards-compatible feature additions or workflow extensions.
- `MAJOR`: breaking CLI, data schema, or API changes.

## [0.1.0] - 2026-02-25

### Added
- Initial public project structure and CLI workflow (`generate`, `analyze`, `sweep`).
- Deterministic run-folder architecture for reproducible scientific ML experiments.
- Boltzmann thermodynamic analysis and temperature sweep outputs.
- Editable packaging via `pyproject.toml` and `src/` layout.

### Documentation
- Added README guidance for installation, quickstart, architecture, and development.
- Added this changelog and release/versioning policy.
